Poemas Interativos

Autor: Ana Silva | Data de Publicação: 15/03/2010

Sobre
Versões
Metadados
Como Acessar

Poemas Interativos é uma obra pioneira na literatura digital brasileira que explora as possibilidades de interação do leitor com o texto poético. Cada poema se transforma e se reconstrói basedo nas escolhas do leitor, criando uma experiência única a cada leitura.

Requisitos Técnicos

Esta obra requer: Flash Player (versões antigas) ou emulador Ruffle (versões modernas). Para a versão de instalação local, é necessário Windows 7 ou superior.

Palavras-chave

poesia digital, narrativa não-linear, ficção interativa, literatura experimental

Histórico de Versões

Esta obra passou por várias versões ao longo do tempo:

Versão 3.0 (2020) - Versão atual, adaptada para HTML5 e JavaScript
Versão 2.1 (2015) - Adicionada compatibilidade com dispositivos móveis
Versão 1.0 (2010) - Versão original em Flash

Detalhes da Versão Selecionada

Versão 3.0 (2020)

Esta versão foi completamente reformulada para utilizar tecnologias modernas após a descontinuação do Flash. Mantém toda a funcionalidade original com melhor desempenho e compatibilidade.

Alterações principais:

  • Migração de Flash para HTML5/CSS3/JavaScript
  • Interface responsiva para dispositivos móveis
  • Melhoria na performance de carregamento
  • Novos poemas adicionados
